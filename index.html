Автоматизация работы с kad.arbitr.ru

На данной странице вы можете протестировать работу сервиса API для получения информации с kad.arbitr.ru. Сервис работает через http запросы (ИНН или номер дела). В ответ на каждый запрос вы получаете JSON с нужной информацией.

По ИНН физ. или юр. лица вы можете получить все дела, в которых это лицо было участником (истцом, ответчиком или третьим лицом).

По номеру дела вы можете получить ИНН, ОГРН, адреса участников; инстанции дела; все события, зарегистрированные по каждой инстанции; ссылки на документы в электронном виде; даты всех судебных заседаний.

Данный сервис может быть интегрирован в любую корпоративную систему или web сайт для проверки контрагентов. С форматом входных параметров и ответа сервера вы можете ознакомиться скачав файл документации.

Наш сервис не имеет никакого отношения к kad.arbitr.ru, а предоставляет вам более удобный способ взаимодействия с информацией, находящейся в открытом доступе на данном сайте.

Для подключения к сервису напишите на email:
support@parser-api.com
